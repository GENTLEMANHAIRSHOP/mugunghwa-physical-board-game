# Mugunghwa Physical Board Game 🌺🎮

Arduino와 센서를 활용해 만든 피지컬 컴퓨팅 보드게임 프로젝트입니다.

한국 전통 놀이인 "무궁화 꽃이 피었습니다"를 아두이노, 서보모터, 센서, LED, 부저를 활용한 인터랙티브 보드게임으로 구현하는 것을 목표로 했습니다.

## 프로젝트 소개

이 프로젝트는 사용자의 움직임을 감지하고, 게임 규칙에 따라 말의 이동과 경고 시스템을 제어하는 피지컬 컴퓨팅 게임입니다.

술래가 "무궁화 꽃이 피었습니다"를 외치는 동안 플레이어는 움직일 수 있고, 술래가 돌아보는 순간에는 움직이면 안 됩니다.

센서가 움직임을 감지하면 경고가 증가하고, 경고가 일정 횟수를 넘으면 탈락 처리됩니다.

## 제작 목적

- 전통 놀이를 아두이노 기반 보드게임으로 재해석
- 센서, 서보모터, LED, 부저를 활용한 피지컬 컴퓨팅 학습
- 랜덤 타이밍과 반칙 판정이 있는 게임 로직 구현
- 학교 발표 또는 전시용 인터랙티브 게임 제작

## 주요 기능

- 랜덤 시간 후 술래가 돌아보는 상태 전환
- 움직임 감지 시 반칙 경고 증가
- 경고 2회까지 허용, 3회째 탈락 처리
- 서보모터를 이용한 말 이동
- LED와 부저를 이용한 상태 표시
- ESP32 또는 디스플레이를 활용한 눈 표시 확장 가능

## 사용 부품

- Arduino Uno
- 서보모터
- 움직임 감지 센서 또는 버튼
- LED
- 부저
- 점퍼선
- 브레드보드
- 보드게임 말 또는 3D 프린팅 말
