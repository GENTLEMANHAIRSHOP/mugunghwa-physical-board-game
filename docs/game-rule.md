# Game Rule

## 게임 이름

무궁화 꽃이 피었습니다 피지컬 보드게임

## 기본 규칙

이 프로젝트는 전통 놀이인 "무궁화 꽃이 피었습니다"를 보드게임과 아두이노 시스템으로 구현한 것입니다.

플레이어는 움직일 수 있는 시간에 앞으로 이동하고, 멈춰야 하는 시간에는 움직이면 안 됩니다.

## 게임 상태

### 1. Safe Time

Safe Time은 플레이어가 움직일 수 있는 시간입니다.

이 시간에는 LED가 켜지고, 서보모터를 이용해 말이 앞으로 이동합니다.

### 2. Freeze Time

Freeze Time은 플레이어가 멈춰야 하는 시간입니다.

이 시간에 센서가 움직임을 감지하면 반칙으로 판정됩니다.

### 3. Warning

움직임이 감지되면 경고가 1회 증가합니다.

경고는 총 2회까지 허용되며, 3회째에는 게임 오버가 됩니다.

### 4. Game Clear

말이 목표 지점에 도달하면 게임 클리어입니다.

### 5. Game Over

경고가 3회가 되면 게임 오버입니다.

## 기본 흐름

```txt
게임 시작
   ↓
Safe Time
   ↓
말 이동
   ↓
Freeze Time
   ↓
움직임 감지 여부 확인
   ↓
경고 또는 계속 진행
   ↓
목표 도달 시 게임 클리어
```
